Overview

The Valero Alamobowl, a yearly NCAA Football Bowl Game, needed an update to their web experience so that it stood out from the other Bowl Games' websites.

The team convened and mapped out a user flow that would benefit the client, not only in the short-term through Bowl Game attendance, but also long-term via membership and year-round event participation.

Upon arrival, the user can immediately see who is playing and buy tickets. When scrolling, the home page provides a web experience that leads the user to information about events, memberships, and San Antonio tourism.

From then on, users can take a much deeper dive into everything the Alamobowl provides using the main menu. This too, is a one-stop area to navigate the entire website.
